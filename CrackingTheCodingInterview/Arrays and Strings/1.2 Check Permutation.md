Pg. 90 (101): Check if two strings are permutations of each other

#### Advice:
- Clarify the alphabetic, ask if case sensitive, ask if whitespace is significant

## Sorting

```c++
class Solution {
public:
	bool check_permutation(string s1, string s2) {
		sort(s1.being(), s2.begin());
		return s1 == s2;
	}
}
```

- O(nlogn + mlogm) Time
- O(1) Space

## HashMap

```c++
class Solution {
public:
	bool check_permutation(string s1, string s2) {
		int count[256] = {};
		for (auto &ch : s1) count[ch]++;
		for (auto &ch : s2) {
			if (count[ch] <= 0) return false;
			count[ch]--;
		}
		return true;
	}
}
```
